## 指令用法
+ 將您的機器人添加為好友或邀請它加入您的群組
+ 您可以使用命令 !搜尋+空一格+縣市+區 ，來獲取旅館的信息。EX:!搜尋 高雄市新興區
+ 您可以使用命令 !地圖+空一格+旅館名字 ，來獲取旅館的地點位置。EX:!搜尋 雅都旅社
+ 您可以傳自己的位置訊息，來獲取附近五公里的旅館。
